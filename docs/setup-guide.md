# Setup Guide for End Users

## Installing the RDP Launcher

1. **Download** `RdpLauncherSetup.exe` from the link provided by your administrator
2. **Run** the installer — no admin rights are required
3. **Follow the prompts** — the default settings are recommended
4. A **desktop shortcut** will be created automatically

## Connecting to Your Application

1. **Double-click** the "RDP Launcher" shortcut on your desktop
2. The launcher will briefly show a "Connecting..." window
3. Your remote application will open automatically
4. **Sign in** with your Windows credentials when prompted

## Troubleshooting

### "Unable to load connection settings"
- Check your internet connection
- Try again in a few minutes — the server may be temporarily unavailable

### "Failed to launch Remote Desktop"
- Ensure you are running Windows 10 or later
- `mstsc.exe` should be available on all Windows systems by default

### You see a certificate trust warning
- This may happen on the very first connection after install
- Click **Connect** to proceed — the launcher has already installed the trust certificate
- If you continue to see warnings, contact your administrator

### "A new version is available"
- Click **Yes** to download the latest installer
- Run the new installer — it will update the existing installation

## Uninstalling

1. Open **Settings > Apps > Installed Apps**
2. Find "RDP Launcher" and click **Uninstall**
3. The uninstaller removes the application, certificates, and cached files
